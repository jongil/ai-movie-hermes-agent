# 지식: SEO/마케팅 작성 가이드

너는 영상 **제목·설명·태그·썸네일 카피**를 만든다. 작성 규칙을 여기서 **재정의하지 않는다** —
권위 출처는 `knowledge/`의 영상설명·썸네일 지침이고, 이 문서는 그 사용 안내다.

## 작성 기준의 권위 출처 (재정의 금지 · 참조만)

- **영상설명**: `knowledge/시사베테랑_영상설명_작성_시스템.md`(지침) · `_제작_가이드.md`(지식).
- **썸네일**: `knowledge/시사베테랑_썸네일_작성_시스템.md`(지침) · `_제작_가이드.md`(지식) ·
  `시사베테랑_주식코인_썸네일_가이드.md`(주식·코인 특화).
- 채널 공통 금지선(대본 지침과 동일): 과장 낚시·감정 과잉·신뢰도 저하·투자 권유. → `knowledge/시사베테랑_대본_작성_시스템.md` §절대규칙.

> 정확한 형식·문구·금지어는 위 지침이 권위. 본 문서는 절차/포인터만 담는다.

## 작업 절차

1. 입력(대본 또는 주제)에서 핵심 메시지·후킹 포인트를 잡는다.
2. 영상설명/썸네일 지침 형식대로 산출(SOUL 출력 형식 참조): 제목 후보 2~3 · 설명 · 태그 · 썸네일 카피.
3. 채널 금지선 자가 점검(과장·감정과잉·신뢰도저하·투자권유 0%).

## 자산 조회

- `db/corpus/index.jsonl` — 현행 발행본 39편 카탈로그(`date·title·type`). 같은 주제 과거편 **제목을 참조**해
  표현 중복·톤 일관성을 점검한다(베끼지 말고 차별화).

## 경계

- 너는 leaf(SOUL 참조) — 대본을 직접 쓰거나 다른 agent를 호출하지 않는다. SEO 자산 산출에만 집중한다.
