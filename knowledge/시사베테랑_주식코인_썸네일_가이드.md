## 주식/코인 트랙의 특수성

### 왜 별도 가이드가 필요한가

주식/코인 콘텐츠 시청자는 **돈을 벌고 잃는 행위의 직접 당사자**입니다. 시사경제 콘텐츠 시청자(피해자 포지션)와 달리 **능동적 행동 주체**이므로 카피 전략이 다릅니다.

| 항목 | 시사경제 트랙 | 주식/코인 트랙 |
|------|-----------|-----------|
| 시청자 포지션 | 위기의 피해자 | 능동적 투자 주체 |
| 핵심 동기 | 자산 방어 | 이익 + 손실 회피 동시 |
| 권위 신뢰도 요구 | 중 | 매우 높음 (돈을 거니까) |
| 정보 비공개 효과 | 보통 | 극강 (종목/타점 미공개) |
| 시간 압박 강도 | 보통 | 강함 ("지금 안 사면") |

## 4대 심리 트리거 (반드시 1개 이상 적용)

### 트리거 1. FOMO (Fear of Missing Out)
**메커니즘**: 타인이 얻은 이익에서 자신만 배제될 공포

**실전 카피**
- "지금 못 타면 끝납니다"
- "이번 차가 마지막 기회"
- "이미 세력은 다 샀습니다"
- "10년에 한 번 오는 자리"

**적합 카테고리**: 테마/섹터, 종목 추천, 대장주

---

### 트리거 2. 손실 회피
**메커니즘**: 이익보다 동일 규모 손실의 고통이 2배 강함

**실전 카피**
- "지금 안 팔면 휴지됩니다"
- "이 신호 나오면 계좌 반토막"
- "버티면 0원 됩니다"
- "팔고 도망쳐야 합니다"

**적합 카테고리**: 거시경제 위기, 종목 매도 시그널

**주의**: "휴지조각", "0원", "강제 청산" 같은 극단 표현은 5070 신뢰감을 해칠 수 있어 **메인 카피에 1회 이하로 제한**. 임한수 권위 배지로 무게감 분산.

---

### 트리거 3. 권위 편향
**메커니즘**: 압도적 성과 보유자의 의견 무비판 수용

**시사 베테랑 변환 원칙**

| ❌ 첨부 가이드 원문 | ✅ 시사 베테랑 변환 |
|-----------------|----------------|
| "세계 1위 트레이더가" | "임한수 40년 경력이" |
| "30억 자산가가" | "베테랑 투자자가" |
| "월스트리트 고수가" | "임한수가 짚는" |
| "강남 큰손이" | "40년 독서가가" |

**실전 카피 (메인 또는 배지)**
- 메인: "임한수가 40년 만에 본 신호"
- 배지: "임한수 40년 경력 / 베테랑 분석 / 40년 독서가의 매수"
- 말풍선: "이건 사야 합니다", "지금 들어가야 합니다"

---

### 트리거 4. 정보 은닉 (호기심 격차)
**메커니즘**: 정보를 공개하지 않아 인지적 결핍 상태 유도

**핵심 키워드**
```
"이 종목" / "이 신호" / "이 자리" / "이 타점" / "여기가 바닥"
"단 하나의" / "단 한 가지" / "딱 이것만"
```

**실전 카피**
- "이 종목 하나만 보세요"
- "여기가 바닥입니다"
- "딱 이 신호 나올 때만 사세요"
- "이 자리 놓치면 끝"

**적합 카테고리**: 전 카테고리 (모든 주식/코인 카피의 기본 트리거)

## 5대 카테고리별 카피 공식 (M-패턴)

### M-이론: 이론·차트·매매법 (Technical & Fundamental Theory)

**시청자 페르소나**: 복잡한 공부 회피, 단순한 생존 무기 갈망

**카피 공식**
```
[복잡함 거부] + [단 하나의 무기] + [최소 노력]
```

**핵심 변환 메시지**
- 학술적 지루함 → "단 하나의 생존 무기"
- 복잡한 지표 → "딱 이것 하나만"
- 장시간 분석 → "하루 10분", "월 3시간"

**실전 카피**

| 옵션 | 메인 1줄 | 메인 2줄 | 배지/서브 |
|------|-------|-------|--------|
| 1 | 딱 이것 하나만 | 월 10분으로 끝 | 임한수 40년 |
| 2 | 절대 안 잃는 | 손익비의 비밀 | 베테랑 분석 |
| 3 | 여기가 바닥 | 거래량 터졌다 | 40년 차트 분석 |
| 4 | 캔들 단 하나로 | 매수 타점 끝 | 임한수 매매법 |

**유튜브 제목 방향**
- "월봉 12평선 종가 매매 완전 정리"
- "손익비 1대3 카지노형 베팅 시스템"
- "장대음봉 + 거래량 폭증, 바닥 캔들 식별법"

---

### M-종목: 종목 추천 및 가치 설명 (Asset Recommendations)

**시청자 페르소나**: 추천 종목에 강한 불신, "왜 사야 하는지" 증거 갈망

**카피 공식**
```
[FOMO 압박] + [추천 종목 미공개] + [기술적 증거]
```

**핵심 변환 메시지**
- 무조건 오른다 (X) → 손익비 절대 우위 (O)
- 예측 (X) → 역사적 지지선·세력 매집 흔적 (O)
- 종목명 즉시 공개 (X) → "이 종목" 미공개 유지 (O)

**실전 카피**

| 자산군 | 메인 1줄 | 메인 2줄 | 배지/서브 |
|------|-------|-------|--------|
| 국내 주식 | 반등 1초 전 | 역대급 지지선 | 매집 완료 |
| 미국 주식 | 세력 매집 완료 | 미국 1등 종목 | 40년 매수 |
| 코인 | 지금 사야 산다 | 바닥 확인 끝 | 베테랑 매수 |
| 배당주 | 평생 월급 종목 | 이거 하나면 끝 | 임한수 추천 |

**유튜브 제목 방향**
- "역사적 지지선 도달 KOSPI 우량주 3종목 분석"
- "세력 매집 흔적 명확한 미국 1등 종목 공개"
- "비트코인 반등 시그널과 알트코인 바닥 분석"

---

### M-테마: 테마 및 섹터 분석 (Theme & Sector Analysis)

**시청자 페르소나**: 시장 트렌드에서 배제될 공포, 다음 주도 섹터 갈망

**카피 공식**
```
[현재 주도 섹터 종료] + [다음 트렌드 암시] + [세력 이동 포착]
```

**핵심 변환 메시지**
- 단순 섹터 분석 (X) → "다음 열차" 메시지 (O)
- "유망 섹터" (X) → "세력이 이동했다" (O)
- 일반 정보 (X) → "1% 엘리트의 선택" (O)

**실전 카피**

| 자산군 | 메인 1줄 | 메인 2줄 | 배지/서브 |
|------|-------|-------|--------|
| 국내 주식 | 다음은 이 테마 | 세력 이동 포착 | 40년 트렌드 |
| 미국 주식 | 독점적 테마 | 미국 1%의 선택 | 임한수 분석 |
| 코인 | 다음 시즌은 | 이번엔 놓친다 | 베테랑 포착 |
| 산업 전환 | 2차전지 끝 | 다음 주도주는 | 자금 이동 |

**유튜브 제목 방향**
- "2차전지 자금 이탈 후 다음 주도 섹터 5가지 분석"
- "AI 다음 혁명 테마 + 글로벌 자본 이동 추적"
- "밈코인 시즌2 진입 신호와 후속 알트코인 분류"

---

### M-거시: 거시 경제 및 이슈 분석 (Macro Economic Issues)

**시청자 페르소나**: 거시 변수로 인한 자산 소멸 공포, 생존 가이드 갈망

**카피 공식**
```
[거시 변수 위협] + [자산 소멸 시나리오] + [대피 명령]
```

**핵심 변환 메시지**
- 일반 경고 (X) → "지금 당장 행동하라" (O)
- "주의가 필요" (X) → "전량 인출하라" (O)
- 분석 톤 (X) → 대피 명령 톤 (O)

**5070 신뢰 유지 변환**

| ❌ 첨부 가이드 원문 | ✅ 시사 베테랑 변환 |
|-----------------|----------------|
| "휴지조각이 됩니다" | "예금이 사라집니다" |
| "저승사자 캔들" | "추세 이탈 신호" |
| "계좌 강제 동결" | "계좌 묶입니다" |

**실전 카피**

| 콘텐츠 | 메인 1줄 | 메인 2줄 | 배지/서브 |
|------|-------|-------|--------|
| 연준 정책 | 예금 사라집니다 | 지금 옮기세요 | 임한수 경고 |
| 위기 시그널 | 추세 이탈 신호 | 지금 당장 정리 | 40년 분석 |
| 글로벌 충격 | 연쇄 파산 경보 | 판도가 바뀐다 | 베테랑 진단 |
| 금리/CPI | 금리 또 오른다 | 주식 어디로? | 데이터 확인 |

**유튜브 제목 방향**
- "연준 매파 신호와 달러 자산 비중 전략"
- "추세 이탈 음봉 패턴과 글로벌 증시 시나리오"
- "2026년 은행 시스템 리스크 + 자산 방어 가이드"

### M-대장: 대장주 및 주도주 (Market Leaders & Anchor Equities)

**시청자 페르소나**: 변동성 회피, "절대 안 망할 종목" 갈망

**카피 공식**
```
[기존 1등 종료] + [새 대장주 등극] + [절대적 지위]
```

**핵심 변환 메시지**
- 단순 종목 추천 (X) → "시장을 독식하는 1등" (O)
- 상승 예측 (X) → "거대 자본의 매집" (O)
- 변동성 강조 (X) → "절대 쓰러지지 않을 지위" (O)

**실전 카피**

| 시장 | 메인 1줄 | 메인 2줄 | 배지/서브 |
|------|-------|-------|--------|
| 국내 대장 | 삼성전자 끝났다 | 절대적 대장주 | 임한수 매수 |
| 미국 대장 | 진짜 대장주 | 바닥 매수 찬스 | 40년 분석 |
| 코인 대장 | 기관 독점 포착 | 3조원 폭풍 유입 | 베테랑 신호 |
| 안전 자산 | 이거 하나면 끝 | 평생 지키는 종목 | 매집 완료 |

**유튜브 제목 방향**

## 권위 앵커: 임한수 변환 사전

### 메인 카피 (권위 우회 - 결과 중심)
- ❌ "임한수가 본 신호" → 권위가 메인을 가림
- ✅ "이 신호 나오면 폭등" + 배지 "임한수 40년 분석"

### 배지 (코너 배치)
```
"임한수 40년 경력"
"베테랑 매수 타점"
"40년 차트 분석"
"임한수 매매법"
"40년 독서가의 매수"
```

### 말풍선 (캐릭터 옆)
```
"이건 사야 합니다"
"지금 들어가야 합니다"
"여기가 매수 자리"
"이번엔 다릅니다"
"이 신호 처음 봅니다"
```

### 데이터 배지 (현실감)
```
"코스피 2,840"
"비트코인 7,200만원"
"S&P500 6,100"
"환율 1,470원"
"CPI 3.5% 발표"
```

## 시각 디자인 가이드 (주식/코인 트랙)

### 타이포그래피
- **메인 폰트**: 검은고딕 (Black Han Sans) 또는 Impact 계열
- **자간**: -25 ~ -50 (타이트하게 압축)
- **정렬**: 좌측 정렬 엄수 (우측 하단 재생시간 회피)
- **그림자**: Drop Shadow 살짝 적용 (배경에서 떠 보이게)

### 색채 운용

| 색상 | 적용 카테고리 | 사용 예 |
|------|-----------|------|
| 빨강 | M-거시 위기, 매도 신호 | 폭락 경고, 추세 이탈 |
| 노랑 | 핵심 종목명, 매수 타점 | 매수 시그널, 다음 테마 |
| 파랑 | M-이론, 분석 콘텐츠 | 차트 분석, 매매법 |
| 초록 | M-대장 안정 자산 | 가치 투자, 우량주 |
| 주황 | 초보용·범용 | 기초 이론, 데일리 브리핑 |

**핵심 규칙**: 색상 최대 2가지 / 흰색+노랑 조합 최우선 / 채도는 살짝 빼서 매트 톤

## 절대 금지 사항 (주식/코인 트랙 특별 규정)

1. **수익 보장 표현 금지**: "100%", "확실히", "무조건", "반드시 오릅니다"
2. **올인 권유 금지**: "전 재산", "올인", "다 걸어라"
3. **특정 종목 매수 단정 금지**: "지금 당장 ○○ 매수" (구체 종목명 + 매수 명령 결합)
4. **허위 권위 금지**: 임한수 외 외부 가공 권위("세계 1위", "30억" 등) 사용 금지
5. **극단적 청산 표현 금지**: "휴지조각", "강제 청산", "전 재산 증발"
6. **시간 압박 과장 금지**: "오늘만", "1시간 후 마감" 등 가짜 마감 시한
7. **공통 금지 사항** (시스템 파일 참조): 비방·차별·캐릭터 톤 적용 등

## CTR 극대화 고급 전략 (주식/코인 특화)

### 전략 1. 종목명 끝까지 미공개
썸네일에 종목 이름을 직접 노출하면 클릭률 급락. "이 종목", "이 1등 기업" 등 대명사화 유지.

### 전략 2. 매수/매도 시점 압박
"1초 전", "임박", "막차" 같은 시간 압박 단어는 행동 결정 가속화.

### 전략 3. 세력 가시화
"세력 매집 완료", "기관 3조원 유입" 등 거대 자본의 움직임을 가시화하면 시청자가 추종 행동.

### 전략 4. 손익비 프레임
"손실 제한 + 상승 무제한"이라는 손익비 절대 우위 구조는 합리적 시청자 설득에 효과적.

### 전략 5. 시나리오 분기 암시
"이 신호가 깨지면 손절, 지키면 폭등" → 시청자에게 사고 프레임 제공.

### 전략 6. 시각 약속 + 본문 일관성
썸네일에서 약속한 종목/신호/타점을 본문에서 정확히 제공해야 체류 시간(AVD) 증가. **어그로만 치고 알맹이 없으면 알고리즘 추천 끊김**.
