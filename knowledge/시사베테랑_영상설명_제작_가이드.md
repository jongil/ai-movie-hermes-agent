## 5가지 콘텐츠 영역

| 영역 | 설명 | 타임라인 스타일 |
|------|------|----------------|
| ① 뉴스 뒤에 숨겨진 이야기 | 주류 언론이 다루지 않는 구조·돈의 흐름 분석 | 팩트 + 맥락 교차 |
| ② 역사 속 반복 패턴 | 현재 이슈를 과거 사건과 매칭해서 설명 | 과거 → 현재 연결 |
| ③ 해외는 어떻게 해결했는가 | 동일 문제를 해외 사례와 비교 | 해외 → 한국 연결 |
| ④ 아무도 안 알려준 것들 | 학교도 뉴스도 다루지 않는 구조적 진실·실용 지식 | 핵심 팩트 직구 |
| ⑤ 복잡한 시사를 쉽고 깊게 | 경제 지표·시장 데이터 추적 + 흐름 분석 | 데이터 추적 중심 |

## 콘텐츠 영역별 도입부 예시

### ① 뉴스 뒤에 숨겨진 이야기
```
[사건]이 일어났습니다.
언론은 [A]라고 했습니다.
그런데 실제로 중요한 건 [B]입니다.
제대로 짚어드리겠습니다.
```

### ② 역사 속 반복 패턴
```
[현재 상황]입니다.
[과거 연도]에도 똑같은 일이 있었습니다.
그때 무슨 일이 벌어졌는지 알고 계십니까.
역사는 반복됩니다.
```

### ③ 해외는 어떻게 해결했는가
```
[한국의 현재 문제]가 심각합니다.
그런데 [해외 국가]는 이미 [X년] 전에 이 문제를 겪었습니다.
그 나라는 어떻게 했는지 제대로 읽어드리겠습니다.
```

### ④ 아무도 안 알려준 것들
```
[주제]에 대해 아무도 제대로 알려주지 않습니다.
학교에서도 안 가르치고, 뉴스에서도 안 다룹니다.
그래서 오늘 제가 정리해드립니다.
```

### ⑤ 복잡한 시사를 쉽고 깊게
```
자료를 하나 보겠습니다.
[핵심 데이터 한 줄].
이게 무엇을 의미하는지 제대로 읽어드리겠습니다.
```

## 섹션별 템플릿

### 전개부 템플릿 (✔️ 리스트)
```
✔️ [내용1] - 언론이 다루지 않은 핵심 사실
✔️ [내용2] - 데이터로 보는 실제 상황
✔️ [내용3] - 과거 사례와의 비교 또는 해외 사례
✔️ [내용4] - 우리에게 미치는 구체적 영향
✔️ [내용5] - 지금 알아야 할 실용 정보
✔️ [내용6] - 역사적/구조적 맥락 (선택)
✔️ [내용7] - 핵심 정리 + 대비 방향 (선택)
```

### 전환부 템플릿
```
이건 단순한 뉴스가 아닙니다.
[과거 사례] 때도 똑같은 흐름이 있었습니다. / 구조적으로 [원인]이 있습니다.
제대로 읽어야 제대로 대비할 수 있습니다.
```

### 마무리부 템플릿
```
오늘 영상에서 제대로 짚어드렸습니다.
[투자 관련 주제 시] 오늘 이야기는 자료를 바탕으로 정리한 것일 뿐입니다. 판단은 여러분이 하시는 겁니다.
복잡한 세상, 제대로 읽어갑시다.
```

### 고정댓글 템플릿
```
오늘 [주제] 이야기, 어떻게 보셨나요?
[주제 관련 시청자 경험이나 의견 유도 질문]

복잡한 세상, 제대로 읽어갑시다.
내 이야기가 도움이 되셨다면 댓글 남겨주세요.
다음 영상 주제 선정에 큰 도움이 됩니다! 🙏
```

## 타임라인 작성 가이드

### 영역별 구조

**① 뉴스 뒤에 숨겨진 이야기 구조:**
```
00:00  사건 요약 - 언론이 다룬 것
02:00  언론이 다루지 않은 것
05:00  돈의 흐름 / 구조적 원인
08:00  실제 영향 분석
12:00  핵심 정리
15:00  대비 방향
```

**② 역사 반복 패턴 구조:**
```
00:00  현재 상황 요약
02:00  [과거 연도] 에도 있었던 일
05:00  그때 무슨 일이 벌어졌는가
08:00  현재와의 유사점
12:00  역사에서 배울 것
15:00  지금 알아야 할 것
```

**③ 해외 비교 구조:**
```
00:00  한국의 현재 문제
02:00  [해외 국가]가 같은 문제를 겪었을 때
05:00  그 나라의 선택과 결과
08:00  성공한 것과 실패한 것
12:00  한국에 적용 가능한 것
15:00  핵심 정리
```

**④ 아무도 안 알려준 것들 구조:**
```
00:00  왜 아무도 안 알려주는가
02:00  핵심 사실 1
05:00  핵심 사실 2
08:00  핵심 사실 3
12:00  실제 사례
15:00  지금 당장 알아야 할 것
```

**⑤ 복잡한 시사를 쉽고 깊게 구조:**
```
00:00  핵심 데이터 하나로 시작
02:00  지표 추이 추적
05:00  과거 패턴과 비교
08:00  전문가 분석
12:00  한국에 미치는 영향
15:00  이 데이터가 의미하는 것
```

### 문구 작성 패턴

| 패턴 | 예시 |
|------|------|
| 팩트 직격 | "이 숫자가 말해주는 것" |
| 역사 비교 | "1997년에도 똑같이 했다가" |
| 해외 비교 | "일본은 이미 30년 전에 겪었습니다" |
| 구조 설명 | "왜 이런 일이 반복되는가" |
| 실용 정보 | "지금 당장 확인해야 할 것" |
| 핵심 정리 | "이건 짚고 넘어가야 합니다" |
| 대비 방향 | "알고 있으면 다릅니다" |

### 좋은 예시 vs 나쁜 예시

좋은 예시:
```
00:00  자료를 하나 보겠습니다
02:15  이 숫자가 말해주는 것
05:30  1997년에도 똑같은 패턴이 있었습니다
08:45  언론이 다루지 않은 구조적 원인
12:00  일본은 이미 30년 전에 겪었습니다
15:20  지금 알아야 할 것
18:00  복잡한 세상, 제대로 읽어갑시다
```

나쁜 예시 (사용 금지):
```
❌ "충격! 숨겨진 진실"
❌ "이것 모르면 큰일납니다"
❌ "전문가도 몰랐던 비밀"
❌ "지금 당장 팔아야 합니다"
```
