## 5070 시니어 타깃 특성과 썸네일 직격 원리

### 왜 직격형 카피여야 하는가

5070 시니어는 다른 세대보다 **결과 명확성**에 강하게 반응합니다.

- **직접 경험 세대**: 1997년 IMF·2008년 금융위기를 몸으로 겪음 → 위기는 "터진다"는 걸 본 세대
- **자산 보유 세대**: 노후 자산·부동산·연금에 민감 → 자기 자산이 직접 호명되면 즉시 반응
- **추론 회피 세대**: 추상명사("구조·패턴·신호")를 보면 멈칫함 → 결과가 즉시 보여야 클릭

따라서 5070 타깃 썸네일은 다음 공식을 따릅니다.

```
[조건] + [내 자산 이름] + [터진다/사라진다/갑니다]
```

### 직격형 vs. 추론형 비교

| 직격형 (5070 적합) | 추론형 (5070 부적합) |
|---------------|----------------|
| 환율 1500원 되면 **국민연금 터집니다** | 환율 천오백원 넘으면 수입 물가 버텨낼 수 없습니다 |
| 2026년 집값 결국 **이 구간으로 갑니다** | 데이터가 말하는 2026년 집값 방향 |
| 한국은 지금 **이상합니다** | 한국 경제, 지금 어디에 와 있는 걸까? |

## 직격 동사 사전 (사용 우선순위)

### 1군 동사 (최우선 - 결과가 즉시 보임)

| 동사 | 적합 주제 | 실전 예시 |
|------|---------|---------|
| 터집니다 | 환율·연금·부동산 | "환율 1500원 되면 국민연금 터집니다" |
| 무너집니다 | 경제·국가·산업 | "2026년 경제가 무너지는 이유" |
| 사라집니다 | 자산·일자리·산업 | "가장 먼저 사라지는 것들" |
| 증발해버린 | 사람·돈·매출 | "성탄절 연말... 증발해버린 사람들" |
| 줄도산 | 기업·산업 | "잘나가던 중국 전기차 갑자기 줄도산" |
| 폭망 / 망해따 | 기업·국가 (서브 톤) | "망해따.." (서브 배지) |
| 갑니다 | 가격·방향 예측 | "결국 이 구간으로 갑니다" |
| 옵니다 | 위기·시점 예측 | "곧 닥칠 가난은 '이 순서'로 옵니다" |
| 끝납니다 | 시대·구조 종료 | "이 시대 곧 끝납니다" |
| 시작됐습니다 | 위기 진행 중 | "경기침체가 시작됐는데" |
| 빠지는 | 함정·늪 | "할부 늪에 빠지는 이유" |
| 떠나는 | 자본·인구 | "외국자본이 한국시장을 떠나는 이유" |
| 패닉 | 충격·반응 | "한국무기 매운맛에 패닉" |

### 2군 동사 (맥락·복합 주제)

| 동사 | 적합 주제 |
|------|---------|
| 이상합니다 / 심상치 않습니다 | 진단·경고 |
| 위험하다 | 상태 진단 |
| 지옥 | 위기 정점 |
| 닥칠 | 임박 위기 |

### 3군 동사 (메인 회피, 서브/제목용)

```
버텨낼 수 없습니다 / 막을 수 없습니다 / 흔들리고 있습니다 / 사라지고 있습니다
```
→ 진행형·간접형은 메인 카피의 직격력을 약화시킵니다.

## 콘텐츠 전략: 실용 베이스 + 감성형 제목

**원칙**: 내용은 실용적으로, 포장은 감성적으로.

| 내용 (실용) | 썸네일 (감성·직격) |
|-----------|----------------|
| 상속 분쟁 예방 3가지 방법 | "**자식들** 싸우는 거 보기 싫으면 지금 해두세요" |
| 일본 고령화 30년 분석 | "일본은 이미 **30년 전**에 겪었습니다" |
| LEI 지표 하락 분석 | "이 신호 나오면 **노후자금 사라집니다**" |
| ETF 포트폴리오 비중 조정 | "이렇게만 해두면 **위기에도 버팁니다**" |

→ **썸네일은 감성 포장 + 직격 결과 담당**, **제목은 실용 정보 담당**으로 역할 분리

## 21개 레퍼런스 분석에서 추출한 직격 핵심 학습

### 학습 1. 환율 + 자산명 + 터집니다 = 최강 시리즈
이미지 4·12·19에서 반복된 공식. 동일 조건(환율 1500원) + 다른 타깃 명사(국민연금/부동산)로 무한 시리즈 가능.

### 학습 2. "결국 이 구간/이 순서" = 정보 독점 암시
이미지 7·11에서 발견. 정답을 알려줄 것 같은 암시로 클릭 욕구 극대화.

### 학습 3. 인용부호 ""는 권위 + 경고 신호
이미지 15·17·21에서 사용. "제 2의 IMF는 이미 시작?", "이거 모르면 평생 가난합니다" 등 전문가 발언처럼 무게감 연출.

### 학습 4. 위기 → 솔루션 2단 구조 = 전환율 최고
이미지 9·15·17에서 검증. 상단(위기 설정) + 하단(솔루션) 구조가 끝까지 보게 만듦.

### 학습 5. "잘나가던 + 갑자기" = 최강 반전 트리거
이미지 13에서 검증. 성공 신화의 갑작스러운 몰락은 가장 강력한 반전 클릭 트리거.

### 학습 6. 96만 조회수 같은 사회적 증거 = 신뢰 폭발
이미지 8에서 검증. "이 정보는 이미 검증됐다"는 인식 형성.

### 학습 7. 실시간 데이터 배지 = 즉각 현실감
이미지 19의 "오늘 환율: 1,470원" 배지처럼 실제 시장 데이터 삽입은 신빙성 폭발.

### 학습 8. 구체 연봉/브랜드명 = 즉각 공감
이미지 20의 "연봉 4천인데 벤츠?"처럼 구체적 수치 + 브랜드명 조합은 공감을 즉각 유발.

```
"팩트 체크 완료"
"40년 독서가의 경고"
```

### 권위 말풍선 (캐릭터 옆)
```
"이건 짚어야 합니다"
"미리 아셨습니까?"
"준비 하셨어?"
"데이터가 이미 나왔습니다"
```

### 실시간 데이터 배지 (현실감 강화)
```
"오늘 환율: 1,470원"
"CPI 삼점오퍼센트 발표"
"LEI 6개월 연속 하락"
"96만 조회수"
```

**규칙**: 권위는 **메인 카피를 강화하는 보조**일 뿐, 메인 카피 자체가 권위 프레임이 되면 안 됩니다.

## CTR 극대화 고급 전략

### 전략 1. 시리즈 확장
동일 조건 + 다른 타깃 명사 = 1주제 → 3~5주제 무한 확장
```
"환율 1500원 되면 국민연금 터집니다"
"환율 1500원 되면 부동산 터집니다"
"환율 1500원 되면 자영업 끝납니다"
```

### 전략 2. 위기 → 구원 2단 완결
```
상단: "곧 지옥의 시작이다"
하단: "2026년 경제 살아남는 방법"
```
공포 설정 + 솔루션 암시 = 끝까지 시청 동기 부여.

### 전략 3. 실시간 데이터 삽입
배지에 "오늘 환율: 1,470원" 같은 현실 데이터 삽입 → 신빙성 폭발.

### 전략 4. 정보 비대칭 구조
"외국인은 이미 알고 있다", "직장인 90%가 모르는" → 시청자를 정보 약자로 포지셔닝.

### 전략 5. 역사 트라우마 환기
"1997년", "30년 전 일본" 등 5070이 직접 경험한 사건 환기.

### 전략 6. 사회적 증거 배지
"96만 조회수" 같은 검증 배지 → "다른 사람도 이미 봤다"는 안전감.

## 직접 비교: ❌ 약한 카피 vs ✅ 직격 카피

| 주제 | ❌ 약한 카피 (간접·추상) | ✅ 직격 카피 (3요소 충족) |
|------|----------------------|-----------------------|
| 환율 위기 | 환율 천오백원 넘으면 수입 물가 버텨낼 수 없습니다 | 환율 1500원 되면 국민연금 터집니다 |
| 경기침체 | 한국 경제, 지금 어디에 와 있는 걸까? | 한국은 지금 이상합니다 |
| 부동산 | 데이터가 말하는 2026년 집값 방향 | 2026년 집값 결국 이 구간으로 갑니다 |
| 미중 패권 | 미국이 압박하자 중국이 꺼낸 카드 | TSMC 흔들리자 칼 꺼내든 삼성전자 |
| 가난 경고 | 경제 위기는 항상 이 순서로 왔습니다 | 곧 닥칠 가난은 '이 순서'로 옵니다 |
| 외국 자본 | 자본 시장의 구조적 신호 | 외국자본이 한국시장을 떠나는 이유 |
| 역사 비교 | 1997년 위기와 닮은 5가지 구조적 이유 | 1997년에도 똑같이 했습니다 |
| 솔루션 | 위기에도 자산 지키는 방법 | 내 돈 지키는 3가지 방법 |

## 절대 하지 말아야 할 카피 (채널 정체성 + 직격성 동시 훼손)

| 금지 카피 | 이유 | 대체 카피 |
|---------|------|---------|
| "환율이 결국 이런 흐름을 보입니다" | 추상명사+간접 동사 | "환율 1500원 결국 옵니다" |
| "수입 물가 버텨낼 수 없습니다" | 추상명사+간접 동사 | "국민연금 터집니다" |
| "데이터가 말하는 신호" | 추상명사 두 개 연속 | "이 데이터, 무시하면 끝납니다" |
| "한국 경제, 어디에 있는 걸까?" | 의문이 약함 | "한국은 지금 이상합니다" |
| "임한수가 40년 만에 본 위기" | 권위가 메인을 가림 | (메인 직격 + 배지 "임한수 40년 분석") |
| "구조적 변화의 시그널" | 추상명사 + 외래어 | "이 변화, 노후자금 사라집니다" |
